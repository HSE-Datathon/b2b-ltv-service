## Сервис LTV и рекомендаций (NBO)

Сервис на FastAPI для расчёта LTV клиентов B2B и выдачи рекомендаций по продуктам (Next Best Offer) на основе обученных моделей CatBoost и KMeans.

### Как запустить

**1. Обучить модели** (из папки `b2b-ltv-analytics`):

```bash
cd b2b-ltv-analytics
python export_models.py
```

Скрипт сохранит артефакты в `b2b-ltv-service/app/models/`.

**2. Запустить сервис:**

```bash
cd b2b-ltv-service
python -m uvicorn app.main:app --reload
```

Сервис будет доступен на [http://localhost:8000](http://localhost:8000).

### Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/` | Веб-интерфейс |
| GET | `/docs` | Swagger UI |
| GET | `/health` | Статус сервиса |
| GET | `/companies` | Список всех компаний с LTV-сегментом |
| POST | `/ltv/predict` | Прогноз LTV для компании |
| GET | `/ltv/segments` | Статистика по LTV-сегментам |
| POST | `/nbo/recommend` | Рекомендации продуктов для компании |
| GET | `/nbo/top` | Топ-N продуктов по NBO-скору |

### Модели

- **CatBoost** — регрессор LTV, предсказывает будущую выручку компании за 6 месяцев
- **KMeans** — кластеризация клиентов (3 кластера) по транзакционному профилю
- **NBO-скоры** — предрасчитаны для каждого кластера на основе популярности и средней выручки продуктов
